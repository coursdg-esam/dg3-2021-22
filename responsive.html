Section

L'élément ‹section› ne doit pas être utilisé comme un conteneur générique : c'est le rôle de ‹div›, notamment lorsque le sectionnement du contenu sert uniquement la mise en forme. Pour savoir lequel utiliser, on peut se demander si la section doit apparaître sur le plan du document : si oui, on utilisera ‹section›, sinon utiliser ‹div›.

Aside

L'élément HTML ‹aside› (en anglais, "aparté") représente une partie d'un document dont le contenu n'a qu'un rapport indirect avec le contenu principal du document.

Redimensionner le navigateur pour changer la mise en page.
